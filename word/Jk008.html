グローバル変数(global variable)

関数の外で定義され、どの関数からも参照できる変数のことをいう。同一コード内部ではもちろん、C言語などでは特定の予約語(extern)を使用することで、他のソースコードからでも使用できる。C言語などのプログラミング言語ではコンパイルの後にリンクという操作を行うが、このリンクの操作が大きく意味を持つ。externという予約語は、「どこか別のソースに定義されている(はずの)変数を、ここでも使いますよ」という曖昧な問題を解決するものである。以下に簡単な例を載せる。

<file1.c>
int ex;//グローバル変数の定義(ここでは実際にメモリ領域を取得)
int main(){
ex = 1;//参照可能
}
int inc(){
ex += 1;//参照可能
}
<file2.c>
extern int ex;//どこかで定義されているexを参照可能にするという意味。
int func2(){
ex += 2;//参照可能
}
